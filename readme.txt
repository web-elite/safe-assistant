A powerful assistant plugin for WordPress that provides security, optimization, and admin tools to simplify your workflow.

== Description ==

Safe Assistant is an all-in-one WordPress assistant plugin that helps you manage, secure, and optimize your website.  
It provides useful features for both administrators and developers, making WordPress easier and safer to use.

**Features:**

* Custom maintenance mode page  
* Security enhancements  
* Admin tools & utilities  
* Integration with Codestar Framework  
* Translation ready (includes `.pot` file)  

This plugin is lightweight, extendable, and developer-friendly.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/safe-assistant` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin settings under **Safe Assistant** in your WordPress dashboard.

== Frequently Asked Questions ==

= Does this plugin work with any theme? =  
Yes, Safe Assistant is theme-independent and works with any WordPress theme.

= Is it translation ready? =  
Yes, `.pot` file is included inside the `languages` directory.
